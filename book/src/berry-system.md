# Berry System

Lazarus uses a Kalos-style berry system with mulch, mutations, and berry club rewards.

- **How it works**: Wailmer Pail and berry supplies come from the Jusmail Town Berry Club. Time of day splits at 7am/7pm.
- **Planting rewards (Berry Club)**:
  - 20 trees: Chilan, Haban, Roseli Berries.
  - 50 trees: Lv 20 Woobat (Chatter, Draining Kiss, Stuff Cheeks, Esper Wing).
  - 100 trees: 10 Amaze Mulch + Shiny Charm.
  - 200 trees: Potted Berry decoration + Trainer Card gold star.
- **Mutation basics**: Only 10 natives grow naturally (Aspear, Cheri, Chesto, Leppa, Lum, Oran, Pecha, Persim, Rawst, Sitrus). Plant pairs adjacent to discover damage-reducing berries:
  - Cheri + Rawst → Occa (Fire) · Chesto + Aspear → Passho (Water) · Oran + Cheri → Wacan (Electric)
  - Chesto + Rawst → Rindo (Grass) · Aspear + Persim → Yache (Ice) · Sitrus + Leppa → Chople (Fighting)
  - Pecha + Oran → Kebia (Poison) · Sitrus + Pecha → Shuca (Ground) · Lum + Rawst → Coba (Flying)
  - Persim + Pecha → Payapa (Psychic) · Leppa + Cheri → Tanga (Bug) · Sitrus + Oran → Charti (Rock)
  - Lum + Pecha → Kasib (Ghost) · Aspear + Lum → Colbur (Dark) · Cheri + Sitrus → Babiri (Steel)
- **Non-mutating berries**: Chilan, Haban, Roseli come only from the 20-tree reward.
